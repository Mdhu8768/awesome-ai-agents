# Contributing to Awesome AI Agents

Thanks for your interest in contributing! This list is maintained by [Groovy Web](https://www.groovyweb.co).

## Guidelines

- Make sure the resource is **actively maintained** (last commit within 6 months)
- Include a brief, factual description (no marketing language)
- Place entries in the correct category alphabetically
- One link per pull request makes reviewing easier
- Check that the link actually works before submitting

## Format

```markdown
- [Name](https://link.com) - Brief description ending with a period.
```

## Quality Standards

- Open-source projects should have 100+ stars (exceptions for genuinely useful niche tools)
- Commercial products should be established (not pre-launch)
- Research papers should be from reputable venues (arXiv, NeurIPS, ICML, etc.)
- Tutorials should be comprehensive and well-maintained

## Not Accepted

- Self-promotion without genuine value
- Duplicate entries
- Dead links or abandoned projects
- Vaporware or pre-alpha projects

## Process

1. Fork the repository
2. Add your resource in the correct section
3. Submit a pull request with a clear description
4. Wait for review (usually within 48 hours)

Questions? Open an issue or email hello@groovyweb.co.
